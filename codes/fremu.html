Emulator Documentation

Overview

The emulator program is designed to simulate the power spectrum of large-scale structures in the universe using neural networks. It provides functionalities to set cosmological parameters, retrieve power spectra, boost functions, and error estimates.

Installation

  1. Clone the repository from GitHub:
    2. git clone https://github.com/your_repo.git
  2. Install the required dependencies:
    3. pip install -r requirements.txt

Usage

Initialization

from emulator import emulator

# Initialize the emulator
em = emulator()

Set Cosmological Parameters

# Set cosmological parameters
em.set_cosmo(Om=0.3, Ob=0.05, h=0.7, ns=1.0, sigma8=0.8, mnu=0.05, fR0=-3e-5)

Get Power Spectrum

# Get power spectrum at redshift z and wave numbers k
power_spectrum = em.get_power_spectrum(k=k_values, z=0.5)

Get Boost Function

# Get boost function at redshift z and wave numbers k
boost_function = em.get_boost(k=k_values, z=0.5)

Get Error Estimate

# Get error estimate at wave number k
error_estimate = em.get_error(k=0.1)

Methods

set_cosmo(Om, Ob, h, ns, sigma8, mnu, fR0)

Set the cosmological parameters for the emulator.

  • Om: Matter density parameter.
  • Ob: Baryon density parameter.
  • h: Hubble constant divided by 100 km/s/Mpc.
  • ns: Scalar spectral index.
  • sigma8: Power spectrum normalization.
  • mnu: Sum of neutrino masses (in eV).
  • fR0: Value of the scalar field f(R) at z=0.

Notes

  • The emulator relies on pre-trained neural network models and cached data stored in the cache directory.
  • Ensure that cosmological parameters are set before retrieving power spectra or boost functions.
